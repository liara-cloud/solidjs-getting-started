# SolidJS getting started
## Run on local machine
```
git clone https://github.com/liara-cloud/solidjs-getting-started.git
```
```
cd solidjs-getting-started
```
```
npm install
```
```
npm run dev
```

## Deploy on Liara
```
npm run build
```
```
cd dist
```
```
liara deploy --platform static
```
